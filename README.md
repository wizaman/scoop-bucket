# wizaman's Scoop Bucket

Personal [Scoop](https://scoop.sh) bucket.

## Installation

Add this bucket to your Scoop installation:

```powershell
scoop bucket add wizaman https://github.com/wizaman/scoop-bucket
```

## Applications

- **[bundletool](https://github.com/google/bundletool)**: Bundletool is a command-line tool to manipulate Android App Bundles.
